Файлы с кодом для практической работы № 2

1. analysis_stats.py
   Скрипт для загрузки данных из файла UN_Tourism_8_9_1_TDGDP_04_2025.xlsx,
   расчёта описательных статистик и выполнения критериев:
   - Шапиро–Уилка;
   - ранговой корреляции Спирмена;
   - ранговой корреляции Кендалла;
   - знаковых рангов Вилкоксона;
   - Манна–Уитни.

2. build_figures.py
   Скрипт для построения графиков, использованных в работе:
   - гистограмма распределения за 2019 год;
   - диаграмма 10 лидеров за 2019 год;
   - диаграмма рассеяния 2019–2021;
   - boxplot для групп SIDS и Non-SIDS.

Как запустить:
1) Поместить рядом со скриптами файл UN_Tourism_8_9_1_TDGDP_04_2025.xlsx.
2) Установить зависимости:
   pip install pandas scipy matplotlib openpyxl
3) Выполнить:
   python analysis_stats.py
   python build_figures.py

Результаты работы analysis_stats.py сохраняются в папку results.
Результаты работы build_figures.py сохраняются в папку figures.
