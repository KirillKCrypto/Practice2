# Practice2
Код для практической работы № 2. 
Файл analysis_stats.py содержит программную реализацию статистической обработки исходных данных. В данном файле выполняются загрузка и подготовка выборок за 2019 и 2021 годы, формирование парной выборки, расчёт описательных характеристик, а также проверка статистических гипотез с использованием критерия Шапиро–Уилка, коэффициента ранговой корреляции Спирмена, коэффициента Кендалла, критерия знаковых рангов Вилкоксона и U-критерия Манна–Уитни.

Файл build_figures.py предназначен для построения графических материалов, использованных в работе. В нём реализовано построение гистограммы распределения показателя за 2019 год, диаграммы десяти стран и территорий с наибольшими значениями показателя, диаграммы рассеяния для сопоставления значений 2019 и 2021 годов, а также boxplot-диаграммы для сравнения групп SIDS и Non-SIDS. Все построенные изображения автоматически сохраняются в отдельную папку.
